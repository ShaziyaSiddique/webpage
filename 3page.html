What is Safari?

It is built into several of Apple's operating systems, including macOS, iOS, iPadOS and visionOS, and uses Apple's open-source browser engine WebKit, which was derived from KHTML. Safari was introduced in Mac OS X Panther in January 2003. It has been included with the iPhone since the first-generation iPhone in 2007. At that time, Safari was the fastest browser on the Mac. Between 2007 and 2012, Apple maintained a Windows version,[5][6] but abandoned it due to low market share. In 2010, Safari 5 introduced a reader mode, extensions, and developer tools. Safari 11, released in 2017, added Intelligent Tracking Prevention, which uses artificial intelligence to block web tracking. Safari 13 added support for Apple Pay, and authentication with FIDO2 security keys. Its interface was redesigned in Safari 15.

Safari development history

Safari's development reflects Apple's commitment to user privacy and innovation. The browser has consistently been a pioneer in adopting new web technologies. For example, Safari was the first major browser to support the HTML5 standard, which was a significant step in web development, allowing for more dynamic and interactive websites. In 2010, Apple introduced the Safari Reader feature, which strips away ads and other distractions from web pages, providing a cleaner reading experience. This was followed by Reading List in 2011, enabling users to save web pages for later, even offline. Safari has also been at the forefront of energy efficiency. Apple's developers have continually optimized the browser to consume less power, extending battery life on MacBooks. This is particularly important for users who rely on their laptops for long periods without access to charging. With each macOS update, Safari receives enhancements that align with the latest operating system features. For instance, the introduction of macOS Big Sur brought a redesigned Safari with a customizable start page, improved tabs, and a new Privacy Report feature that gives users insights into how websites use their data.

Safari's close integration with the Apple ecosystem means it works seamlessly with other Apple services and devices. Features like Handoff allow users to continue browsing on different Apple devices effortlessly. Additionally, Safari supports strong password generation and storage, synchronizing with iCloud Keychain to keep passwords secure across all Apple devices. Apple's continuous focus on privacy is evident in features like the Privacy Report and prevention of third-party tracking. These initiatives make Safari one of the most privacy-focused browsers available. Safari has not only shaped web browsing on Apple devices but also influenced the broader web ecosystem through its innovative features and open-source WebKit engine. Features

Until Safari 6.0, it included a built-in web feed aggregator that supported the RSS and Atom standards. Current features included Private Browsing (a mode in which the browser retains no record of information about the user's web activity),[146] the ability to archive web content in WebArchive format, the ability to email complete web pages directly from a browser menu, the ability to search bookmarks, and the ability to share tabs between all Mac and iOS devices running appropriate versions of software via an iCloud account.[147]

Web compatibility

In Safari's early years, it pioneered several HTML5 features that are now standard, such as the Canvas API.[148] In 2015, Safari was criticized for failing to keep pace with some modern web technologies.[149]

Intelligent Tracking Prevention

In September 2017, Apple announced that it would use artificial intelligence (AI) to reduce the ability of advertisers to track Safari users as they browse the web. Cookies used for tracking will be allowed for 24 hours, then disabled, unless the AI system judges that the user wants to keep the cookie.[150] Major advertising groups objected, saying it will reduce the free services supported by advertising, while other experts praised the change.[151]

Plugin support

Apple used a remotely updated plug-in blacklist to prevent potentially dangerous or vulnerable plugins from running on Safari. Initially, Flash and Java contents were blocked on some early versions of Safari. Since Safari 12, support for NPAPI plugins (except for Flash) has been completely dropped. Safari 14 finally dropped support for Adobe Flash Player.
